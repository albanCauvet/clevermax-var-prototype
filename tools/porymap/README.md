# Porymap Integration

## Usage cible

- Ouvrir le repo `pokeemerald` avec Porymap
- Creer les maps `CM_*`
- Renseigner:
  - metatiles
  - collisions
  - events objets
  - warps

## Convention palette

- Palette chaude mediterraneenne
- Contraste lisible sur ecran GBA
- Limiter les variations inutiles de couleurs
